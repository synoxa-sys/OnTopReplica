# Release Title

OnTopReplica v3.5.1-jp1

# Release Notes

Unofficial personal Japanese-localized build of OnTopReplica v3.5.1.

Changes in this build:

* completed and polished the existing Japanese UI translation,
* fixed the language selector so Japanese displays a Japan flag icon,
* rebuilt the installer as `setup.exe` and `OnTopReplica.msi`.

Notes:

* this is not an official release from the original author,
* the application itself is based on the original `LorenzCK/OnTopReplica` project,
* installer warnings related to old .NET prerequisite settings were left unchanged from the legacy project configuration.
